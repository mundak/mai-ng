# Task Progress

This document tracks the completion status of every task across all milestones.
**Agents must update this file after finishing each task** — mark the task as
complete, fill in the PR link, and record the date.

> **Rule:** When you finish a task, open this file, change the status from
> `pending` to `done`, add the PR number, and commit the update as part of
> your task PR.

---

## How to Update

1. Find your task row in the table below.
2. Change **Status** from `pending` to `done`.
3. Add your **PR** number (e.g. `#12`).
4. Add the **Date** in `YYYY-MM-DD` format.
5. Commit the change in the same PR that delivers the task.
